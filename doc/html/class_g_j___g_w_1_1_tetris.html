Detailed Description

Classe déterminant le fonctionnement d'une partie de Tetris.

Elle fournit à la vue les données du modèle nécessaires à son bon fonctionnement et implémente Subject.

void GJ_GW::Tetris::next ( )

Méthode lançant une nouvelle itération du jeu.

Une itération de Tetris comprend un mouvement automatique de la Bric courante vers le bas ou la génération d'une nouvelle brique courante si la précédente ne pouvait plus descendre.
